2. Action are JS object which have type property and some payload
	{
		type:"changeMyName",
		payload:"Sunil"
	}
3. Action Creators
	Are some function which return action object
	
	let actionCreator = ()=>{
				
					return {
						type:"changeMyName",
						payload:"Sunil"
					}

			       }
	
4. Reducers
	Are some function with accept oldState,action and return newState

	let reducer1 = (oldState,action)=>{ 
				return newState
			}
